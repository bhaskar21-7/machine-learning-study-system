# Statistics & Machine Learning Study System

A growing collection of **visual Statistics and Machine Learning study notes** focused on intuition, structured explanations, mathematical understanding, and clean learning systems.

This repository is being built gradually according to my learning pace and study journey.

## Currently Available

### Statistics Study System

A complete visual statistics learning system covering:

* Data Tables & Frequency
* Bar Graphs & Pie Charts
* Histograms
* Measures of Central Tendency
* Measures of Spread
* Outliers
* Z-Score
* Normal Distribution
* Covariance & Correlation
* Probability
* Bayes' Theorem
* Descriptive vs Inferential Statistics
* Hypothesis Testing
* T-Test

The notes are designed with:

* visual explanations,
* intuition-first learning,
* exam-focused summaries,
* ML/DS relevance,
* and beginner-friendly structure.
