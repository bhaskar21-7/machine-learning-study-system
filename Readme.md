# About This Repository

This repository contains structured study material focused on:

* Statistics for Machine Learning
* Mathematical intuition
* Visual explanations
* Exploratory Data Analysis (EDA)
* Machine Learning fundamentals
* Data preprocessing & feature engineering
* Probability & hypothesis testing
* Revision systems & cheat sheets

The goal is to make difficult concepts:

* easier to revise,
* visually cleaner,
* mathematically intuitive,
* and more practical for ML/DS applications.

# Why This Repository Exists

Most study material online is either:

* overloaded with theory,
* poorly organized,
* tutorial-heavy without intuition,
* or difficult to revise efficiently.

This project focuses on creating:

* structured learning systems,
* concise but meaningful explanations,
* visual understanding,
* and statistics-first ML learning.
